# Stopwatch Web Application

## Overview

This project is a responsive Stopwatch Web Application developed using **HTML**, **CSS**, and **JavaScript**. It provides accurate time tracking through an intuitive and user-friendly interface.

## Features

* Start the stopwatch
* Pause the stopwatch
* Reset the stopwatch
* Record lap times
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript
